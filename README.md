# SETP - Simulated Electronic Trading Platform 📈

SETP provides a sandboxed environment for simulated asset trading, focusing on data consistency and real-time performance.

* **Frontend:** A React 18 frontend built with Vite and TypeScript. It features a high-density, dark-mode "terminal" aesthetic optimized for real-time data feeds and rapid order execution.
* **The Core:** A Django 4.2+ backend providing a robust REST API for portfolio management, order matching, and secure authentication.
* **The Data Layer:**
    * **PostgreSQL:** Handles persistent financial records, holdings, and historical audits.
    * **Redis:** Facilitates real-time price caching and acts as an ingestion point for orders to mitigate race conditions.

## Getting Started

### 1. Prerequisites
Ensure you have the following services active on your local machine:
* **Python 3.10+**
* **Node.js 18+**
* **PostgreSQL** (Port: 5432)
* **Redis Server** (Port: 6379)

### 2. Backend Installation
1.  Navigate to the backend directory and initialize a virtual environment:
    ```bash
    cd backend
    python -m venv venv
    source venv/bin/activate  # Windows: venv\Scripts\activate
    ```
2.  Install dependencies and apply database migrations:
    ```bash
    pip install -r requirements.txt
    python manage.py migrate
    ```
3.  Start the API server:
    ```bash
    python manage.py runserver
    ```

### 3. Frontend Installation
1.  Navigate to the frontend directory and install packages:
    ```bash
    cd frontend
    npm install
    ```
2.  Configure environment variables by creating a `.env` file in the root:
    ```env
    VITE_API_URL=http://localhost:8000
    ```
3.  Launch the development server:
    ```bash
    npm run dev
    ```

### 4. Running the Market Engine
To simulate live market price movements using the integrated dataset, execute the following command in a separate terminal:
```bash
python manage.py run_market_engine
```
